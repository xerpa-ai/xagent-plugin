## Description

Buzhai is an AI-powered robotic cat (v1.0 hardware exists) designed for 25-45yo women living alone. As a companionship-focused agent, it serves as an emotional anchor and encourages users to reconnect with the world. Technically, it integrates OKX DeFi data (real-time APY/TVL from DeFiLlama) for financial wellness features, supports OKX Wallet & Phantom for on-chain interactions, and uses LLM-driven conversations to provide empathetic companionship and behavioral nudges toward social re-engagement.
