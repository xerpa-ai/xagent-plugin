## Description

Autonomous rug pull detection and exit agent built on OKX OnchainOS. Monitors tokens for 5 on-chain signals (dev wallet dumps, smart money exits, holder concentration, liquidity withdrawal, trade flow toxicity), computes a composite RugScore, and auto-exits positions via OKX Agentic Wallet when thresholds are breached.
